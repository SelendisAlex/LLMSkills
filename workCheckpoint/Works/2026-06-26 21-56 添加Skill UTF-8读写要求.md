# 添加 Skill UTF-8 读写要求

- 按用户要求在 `archsave/SKILL.md` 和 `Skills/checkpointsave/SKILL.md` 中添加“使用UTF-8编码读取或修改”的约束。
- 遵守“Skill 文件始终使用英文修改”的长期规则，实际加入英文句子：`Read or modify files using UTF-8 encoding.`。
- 插入位置均为 `Mandatory Startup Check` 的前置说明之后、编号步骤之前，作为读取或修改记录前的通用前置条件。
- 使用 `PYTHONUTF8=1` 分别运行 `quick_validate.py .\archsave` 和 `quick_validate.py .\Skills\checkpointsave`，结果均为 `Skill is valid!`。
- 运行 `rg -n "\p{Han}" .\archsave\SKILL.md .\Skills\checkpointsave\SKILL.md`，目标 Skill 文件未发现中文字符。
